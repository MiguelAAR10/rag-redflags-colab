# Spec 002 — MVP Scope

## Incluido

- Ingesta manual de documentos.
- Chunking.
- Embeddings.
- Retrieval con metadata.
- Respuesta con citas.
- Verificación básica de evidencia.
- API simple.
- Dashboard mínimo de ejecuciones.
- CAVELOG y progress.

## No incluido inicialmente

- Multi-tenant avanzado.
- Billing.
- MCP externo en producción.
- Autonomía total.
- Scraping masivo.
- Acciones irreversibles.

## Métrica de éxito

- 10 preguntas de prueba.
- 90% de respuestas con fuente correcta.
- 0 respuestas sin evidencia cuando la evidencia no existe.
